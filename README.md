# WeatherMX
[WeatherMX](https://github.com/williambargent/WeatherMX) - an elegant, free, open-source web application built to display data form [CumulusMX](http://sandaysoft.com/forum/viewforum.php?f=27) which records values from USB weather stations.

This project is licensed under [GNU AGPLv3](https://choosealicense.com/licenses/agpl-3.0/#).

<https://WeatherMX.bargent.co.uk>
